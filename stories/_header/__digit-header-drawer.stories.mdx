# DigitHeaderDrawer

DigitHeaderDrawer is just the same as `DigitHeader`, but it has support for a drawer. This should really only be used if you have a lot of navigation. Try to instead use `DigitTabs` if there's minimal navigation. 

This demo uses `DigitGammaActionsDummy`, but in your application with Gamma you should use `DigitGammaActions` instead.
