# DigitAutocompleteSelectMultiple

Input to select multiple values via autocomplete when searching for options.

## AutocompleteMultiple

<Canvas>

            const options = [

                { text: "Das Boot", value: "das", disabled: true }

                <DigitAutocompleteSelectMultiple

                    lowerLabel="lowerlabel"
