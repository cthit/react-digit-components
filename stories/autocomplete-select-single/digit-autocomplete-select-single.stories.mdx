# DigitAutocompleteSelectSingle

Input to select a single value via autocomplete when searching for the option.

## AutocompleteSingle

<Preview>
    <Story name="AutocompleteSingle normal">

            const options = [
                { text: "The Dark Knight Rises", value: "the_dark" },

                { text: "Das Boot", value: "das", disabled: true }

            return (
                <DigitAutocompleteSelectSingle
                    size={{width: "224px"}}
                    upperLabel="upperlabel"
                    lowerLabel="lowerlabel"
                    options={options}
