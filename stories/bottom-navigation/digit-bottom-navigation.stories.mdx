# DigitBottomNavigation

Mobile navigation, with tabs that has icons and labels. This should only be used for mobile. If you want similar tabs on desktop or tablet, use `DigitTabs`.

## DigitBottomNavigation without labels

<Preview>
    <Story name="DigitBottomNavigation without labels">
        {() => {
            const [selected, setSelected] = useState(0);
            return (
                <DigitBottomNavigation
                    tabs={[
                        {label: "Home", icon: <Home/>},
                        {label: "Calendar", icon: <Date/>},
                        {label: "Information", icon: <Info/>},
                    ]}
                    onChange={n => setSelected(n)}
                    selected={selected}
                    showLabels={false}
                />
            );
        }}
    </Story>
</Preview>
