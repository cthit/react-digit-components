# DigitDisplayData

`DigitDisplayData` is a easy way to display two columns of data. The first column is usually the column that for every row has a describing text about the value on the same row. E.g. the first column maybe have the texts ["Firstname", "Surname", "Username"], and the second column has ["Smurf", "Smurfsson", "Coder"].

## DisplayData
