# DigitStepper

A useful way to tell a user where they are during a multistep process. Be aware of using to many steps because of mobile users.

<Canvas>
    <Story name="Normal">
        {() => {
            const [activeStep, setStep] = useState(0);
            return (
                <div>
                    <DigitStepper
                        activeStep={activeStep}
                        steps={[
                            {
                                text: "First step of this stepper"
                            },
                            {
                                text: "The second of the cool cooler stepper"
                            },
                            {
                                text: "Whaaaaaaat"
                            },
                            {
                                text: "Almost finished"
                            },
                            {
                                text: "Finished"
                            }
                        ]}
                    />
                    <Center>
                        <DigitButton
                            primary
                            raised
                            text="Next"
                            onClick={() => {
                                setStep((activeStep + 1) % 6);
                            }}
                        />
                    </Center>
                </div>
            );
        }}
    </Story>
</Canvas>
