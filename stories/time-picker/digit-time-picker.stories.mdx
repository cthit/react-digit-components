# DigitTimePicker

A input field with a dialog popup where you can select the time. 

## TimePicker

<Preview>
    <Story name="Normal">
        {() => {
            const [date, setState] = useState(null);
            return (
                <DigitTimePicker
                    upperLabel="Upper label"
                    lowerLabel="Lower label"
                    value={date}
                    onChange={e => setState(e.target.value)}
                />
            );
        }}
    </Story>
</Preview>
